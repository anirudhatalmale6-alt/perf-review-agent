# Performance Review Agent

This project contains a Claude Code custom agent for automated performance code review.

## Agent

- perf-review-agent.md - Scans code for performance bottlenecks across 6 categories: Algorithmic, Memory, I/O & Database, Concurrency, Resource Management, and Serialization/Network

## Usage

Select "Performance Review Agent" from the agent picker in Claude Code, then:
- Paste code or file paths to review
- Ask to scan git diffs for regressions
- Request analysis of specific directories or modules

## Supported Languages

Java/Spring Boot, Python/Django/FastAPI, TypeScript/Node.js, C#/.NET
